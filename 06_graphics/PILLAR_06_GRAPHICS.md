## Core Use Cases

### UC-01: Brand Identity System
**Category:** REFERENCE | BID | SYS  
**Priority:** CRITICAL

**Objective:** Define and maintain consistent brand identity

**Brand Elements:**
```
┌─────────────────────────────────────────────────────────┐
│                 BRAND IDENTITY SYSTEM                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  LOGOTYPE                                               │
│  ├── Primary logo (full)                               │
│  ├── Stacked version                                    │
│  ├── Horizontal version                                 │
│  ├── Icon/wordmark only                               │
│  ├── Minimum sizes (digital: 50px, print: 1")          │
│  └── Clear space requirements                          │
│                                                         │
│  COLOR SYSTEM                                          │
│  ├── Primary colors (2-3)                             │
│  ├── Secondary colors (2-3)                            │
│  ├── Neutral colors (grays, black, white)              │
│  ├── Accent colors (1-2)                              │
│  └── Color codes (HEX, RGB, CMYK, PMS)                │
│                                                         │
│  TYPOGRAPHY                                            │
│  ├── Headline font (1-2)                              │
│  ├── Body font (1-2)                                  │
│  ├── Data/tabular font (optional)                     │
│  └── Font weights and hierarchy                        │
│                                                         │
│  PHOTOGRAPHY STYLE                                     │
│  ├── Lighting (natural, consistent)                    │
│  ├── Subjects (real people preferred)                 │
│  ├── Settings (on-location, authentic)                 │
│  └── Treatment (minimal filters)                       │
│                                                         │
│  ICONOGRAPHY                                           │
│  ├── Style (outline, filled, consistent)              │
│  ├── Size standards                                    │
│  └── Usage guidelines                                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Category:** REFERENCE | AST | LIB  
**Priority:** HIGH

**Objective:** Centralize and organize all visual assets

**Asset Categories:**
```
IMAGES
├── Photography (people, vehicles, facilities)
├── Stock photography (licensed)
├── Graphics/illustrations
└── Icons and logos

VIDEO
├── Commercials
├── Testimonials
├── Vehicle showcases
└── Social clips

TEMPLATES

├── Email templates
├── Digital ads
└── Print materials

BRAND ELEMENTS
├── Logos (all formats)
├── Color swatches
├── Font files
└── Icon sets
```

### UC-04: Print Materials
**Category:** TEMPLATE | PRT | MTR  
**Priority:** MEDIUM

**Objective:** Create professional print collateral

**Standard Print Items:**
- Business cards
- Letterhead
- Envelopes
- Brochures (vehicle, service)
- Postcards (mailers)
- Signage (interior, exterior)
- Presentation folders
- Sell sheets

**Print Specifications:**
```
COLOR MODE: CMYK (process), PMS (spot)
RESOLUTION: 300 DPI minimum
BLEED: 0.125" (1/8") standard
TRIM LINES: Clearly marked
FONTS: Outlined or included
```

**Related Documents:**
- `06-GFX-TEMPLATE-PRT-Brochure-v1.indd`
